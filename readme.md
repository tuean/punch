
### punch

a tiny blog system.


#### plugins

* netty
* spring(minimal)
* guava
* jackson
* reflections
* commons-beanutils
* svelte
* sveltekit-tailwind-blog-starter

#### usage
```shell
cd ./website/sveltekit-tailwind-blog-starter

npm install

npm run dev
```

#### deploy
```shell
cd ./website/sveltekit-tailwind-blog-starter

npm run build

cd archive

node build
```
